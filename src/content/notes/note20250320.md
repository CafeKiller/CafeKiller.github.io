如何将单个数值转换成数组

如: `5 ---> [1, 2, 3, 4, 5]`

```javascript
// Array.from
const number = 5;
const numArr = Array.from({ length: number }, (_, i) => i+1);

// 运算符展开
const number = 5;
const numberArr = [...Array(number).keys()].map(x => x+1);
```
